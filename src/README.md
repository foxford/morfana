# Morfana

Library for morphemes higlighting specializing in Russian language.

## Syntax

| Markdown prefix | Morpheme |
| --------------- | -------- |
| pr              | prefix   |
| su              | suffix   |
| ro              | root     |
| en              | ending   |
| st              | stem     |

Word description looks like this:

```js
ro:1-5;en:6-8;st:1-5;
```
